# Pendahuluan

`Andev Dashboard` adalah web app yang digunakan untuk memonitoring data dari aplikasi Andev (Analytical Development) pada perusahan PT. Genero Pharmaceuticals. Dengan adanya aplikasi ini diharapkan dapat mempermudah dalam memonitoring data yang semula dilakukan secara manual dengan menggunakan excel menjadi web app yang dapat diakses oleh semua pihak yang berkepentingan.

## Stack apa saja yang digunakan?

Project `Andev Dashboard` menggunakan instalasi dari [sini](https://techvblogs.com/blog/how-to-install-bootstrap-5-in-laravel-9-with-vite) dan beberapa stack, diantaranya:

- Laravel 9 + Vite 4.0
- Laravel UI
- Bootstrap 5.2.3
- Blade template engine

Beberapa tools lainnya yang digunakan :

- ECharts 5.4.2
- Datatables

## Cara install

1. Clone repository [andev dashboard](https://github.com/muchfahmiarif/andev-dashboard)
2. Pilih branch `main`
3. Jalankan perintah `composer install`
4. Jalankan perintah `npm install`
5. Copy `.env.example` menjadi `.env`
6. Generate app key dengan perintah `php artisan key:generate`
7. Migrate database dengan perintah `php artisan migrate`
8. Run project dengan perintah `php artisan serve`dan `npm run dev`
