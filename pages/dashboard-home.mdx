# Home

## Interface

Tampilan awal dari aplikasi ini adalah sebagai berikut:

![Dasboard Home](/dashboard-home.png)

Berikut bagian - bagian pada halaman homepage dashboard :

- Pada bagian atas ada search bar, ketika diketik dan ditekan enter user akan diarahkan ke tab baru dan menampilkan hasil pencarian dari kata kunci yang dimasukkan.
- Pada bagian pojok kanan atas terdapat menu informasi yang terdiri dari

  ![informasi home dashboard](/informasi-home-dashboard.png)

  Urutan informasi tersebut adalah sebagai berikut:

  1. Angka tersebut menunjukkan jam saat ini
  2. Icon home yang akan mengarahkan user ke halaman portal
  3. Icon tanda tanya yang akan mengarahkan user ke halaman bantuan
  4. Icon orang yang akan memunculkan menu dropdown yang berisi menu logout dan menu profil
